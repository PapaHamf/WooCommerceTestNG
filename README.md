## General info
This a TestNG framework written in Java with Selenium. It contains various
test scenarios testing the basic functionality of the WooCommerce application (Selenium Demo).
The test suite contains the smoke tests such as user registration, user login, adding 
products to cart, cart widget functionalities and check out process.
Project uses Page Object Pattern (POP). Various test employ the DDT approach with data 
generated by faker library or read from CSV files.

## Technologies
Project is created with:
* Java 21 (JDK 21.0.7)
* Selenium Java 4.32.0
* Webdriver Manager 6.0.1
* TestNG (framework)
* log4j-core 2.25.1
* extentreports 5.1.2
* opencsv 5.12.0
* javafaker 1.0.2

## Setup

Clone the repository:
```
$ git clone https://github.com/PapaHamf/WooCommerceTestNG.git
```

To run the tests:

```
$ mvn clean test
```
